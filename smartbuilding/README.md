# SmartBuilding

Para correr a aplicação é necessário baixar o xampp ou então aceder ao terminal e php -S localhost:8000 e depois no browser correr: http://localhost:8000.
Para conseguir aceder ao serviço é necessário logar com a conta AG1 ou AG2.
A password está predefinida em todas as contas como 1234 para facilitar os testes, mas pode ser alterada na opção no dashboard : alterar password.
